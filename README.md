# IOT-Huzzah-ESP8266-Arduino
Connect Huzzah ESP8266 and send data in real-time by MQTT protocol

https://learn.adafruit.com/adafruit-feather-huzzah-esp8266/using-arduino-ide


1. OPEN ARDUINO IDE

2. SET Additional Board Manager URLs on PREFERENCES: 
http://arduino.esp8266.com/stable/package_esp8266com_index.json 

3. OPEN BOARD MANAGER AND INSTALL ESP8266

4. SET Adafruit Feather HUZZAH ESP8266 Board

5. SET CPU Frequency on 80 MHz & SET Upload Speed to 115200
